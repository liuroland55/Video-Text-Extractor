游 Quick Start Guide
  1. Choose one or more OCR services to configure
  2. Get API keys following the tutorials
  3. Enter keys in settings and test connection
  4. Save configuration and start using
游댯 Azure Vision Tutorial
  1. Visit Azure Website
  2. Register Azure account and login
  3. Create "Computer Vision" resource in Azure Portal
  4. Select "F0" free pricing tier
  5. Find "Keys and Endpoint" in resource page
  6. Copy Key 1 and endpoint URL to settings page
游릭 Google Vision Tutorial
  1. Visit Google Cloud Console
  2. Create new project or select existing project
  3. Enable "Cloud Vision API"
  4. Create service account key
  5. Download JSON key file and copy API key
  6. Enter API key in settings page
游릮 Mathpix Tutorial
  1. Visit Mathpix Signup
  2. Fill registration info and create account
  3. Login and go to dashboard
  4. Find App ID and App Key in "API Keys" page
  5. Copy these two values to settings page
游 ZAI Smart OCR Tutorial
  1. Visit ZAI Website
  2. Register account and login
  3. Create OCR application in console
  4. Get API key and endpoint URL
  5. Apply ZAI template in custom API configuration
  6. Fill in your API key and test connection
