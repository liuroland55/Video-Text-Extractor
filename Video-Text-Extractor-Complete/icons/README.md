# 图标文件目录

这个目录包含插件的图标文件。

## 所需图标文件

- `icon16.png` - 16x16像素图标（工具栏显示）
- `icon48.png` - 48x48像素图标（扩展管理页面）
- `icon128.png` - 128x128像素图标（商店展示）

## 图标设计建议

- 使用简洁的设计风格
- 主要颜色推荐：蓝色 (#007acc) 或紫色主题
- 体现"文本提取"或"视频分析"的概念
- 确保在深色和浅色背景下都清晰可见

## 临时解决方案

在开发阶段，可以使用以下在线工具生成临时图标：

1. [Favicon.io](https://favicon.io/) - 生成多尺寸图标
2. [Canva](https://www.canva.com/) - 在线设计工具
3. [Flaticon](https://www.flaticon.com/) - 免费图标资源

## 最终图标设计

建议使用以下元素组合：
- 摄像机或播放按钮（代表视频）
- 文本行或字母（代表文本提取）
- 魔法棒或扫描效果（代表OCR技术）

图标应该专业、现代且易于识别。
