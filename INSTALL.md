# Video Text Extractor 安装指南

## 📋 安装前准备

### 系统要求
- **浏览器**: Microsoft Edge 88+ 或 Chrome 88+
- **操作系统**: Windows 10+, macOS 10.14+, Linux
- **网络**: 稳定的互联网连接（用于OCR API调用）

### 所需账户
- 至少一个OCR服务账户（推荐Azure Vision，免费额度充足）
- （可选）Google账户、Mathpix账户

## 🚀 安装方法

### 方法一：开发者模式安装（推荐用于测试）

#### 步骤1：下载插件文件
```bash
# 如果从Git仓库下载
git clone https://github.com/your-repo/video-text-extractor.git

# 或直接下载ZIP文件并解压
```

#### 步骤2：在Edge中启用开发者模式
1. 打开Edge浏览器
2. 访问 `edge://extensions/`
3. 开启右上角的"开发人员模式"开关

#### 步骤3：加载插件
1. 点击"加载解压缩的扩展"
2. 选择插件文件夹 `video-text-extractor`
3. 确认加载成功

#### 步骤4：验证安装
- 在工具栏看到插件图标
- 点击图标显示弹出界面
- 确认没有错误提示

### 方法二：商店安装（正式版本）

1. 访问Microsoft Edge加载项商店
2. 搜索"Video Text Extractor"
3. 点击"获取"按钮
4. 等待安装完成

## ⚙️ 首次配置

### 步骤1：打开设置页面
1. 点击插件图标
2. 点击"API设置"按钮
3. 或右键插件图标选择"选项"

### 步骤2：配置OCR服务

#### 推荐配置Azure Vision（最佳性价比）
1. 访问 [Azure官网](https://azure.microsoft.com)
2. 注册Azure账户（需要信用卡验证，但免费额度不会扣费）
3. 创建新资源：Computer Vision
4. 选择定价层：F0（免费）
5. 选择区域：East US 或 West US
6. 创建资源
7. 在资源页面找到：
   - 密钥1（API密钥）
   - 终结点URL
8. 在插件设置页面：
   - 启用Azure Vision API
   - 填写API密钥和终结点
   - 点击"测试连接"验证

#### 备用配置选项

**Google Cloud Vision**
- 免费额度：每月1000次
- 新用户$300免费额度
- 需要信用卡验证

**Mathpix**
- 专长：数学公式识别
- 免费额度：每月1000次
- 推荐用于学术视频

**OCR.space**
- 免费额度：每月25000次
- 无需注册即可使用
- 作为备用选项

### 步骤3：保存配置
1. 点击页面底部的"保存所有配置"
2. 确认显示"配置已保存"提示
3. 关闭设置页面

## 🔧 故障排除

### 常见问题

#### 问题1：插件图标不显示
**解决方案**：
1. 检查是否正确加载插件
2. 确认没有禁用插件
3. 重启Edge浏览器
4. 重新加载插件

#### 问题2：OCR识别失败
**解决方案**：
1. 检查网络连接
2. 验证API密钥是否正确
3. 确认API配额未用完
4. 尝试切换到备用OCR服务

#### 问题3：视频帧捕获失败
**解决方案**：
1. 确保视频已暂停
2. 检查视频是否为HTML5 video元素
3. 尝试刷新页面
4. 检查浏览器控制台错误信息

#### 问题4：权限问题
**解决方案**：
1. 检查插件权限设置
2. 重新安装插件
3. 检查浏览器安全策略

### 错误代码说明

|---------|------|----------|
| OCR_001 | API密钥无效 | 检查密钥输入 |

## 📚 使用技巧

### 提高识别准确率
1. **选择合适帧**：选择文本清晰、对比度高的帧
2. **避免模糊**：暂停在静态画面，避免运动模糊
3. **合适分辨率**：720p以上效果最佳
4. **多OCR服务**：配置多个服务以提高成功率

### 快捷键使用
- `Ctrl+Shift+V`: 快速激活提取
- `Ctrl+C`: 复制全部文本
- `Ctrl+S`: 导出文本文件

### 最佳实践
1. **定期检查配额**：监控API使用情况
2. **备用服务**：配置至少2个OCR服务
3. **格式化开启**：对于学术视频启用智能格式化
4. **历史记录**：定期清理不必要的历史记录
